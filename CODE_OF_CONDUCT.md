# Code of Conduct

This project follows the **Contributor Covenant 2.1**:

→ Full text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Reports of unacceptable behaviour may be sent to **conduct@codelake.dev**. All reports are handled confidentially. We commit to:

- Acknowledging receipt within 72 hours
- Investigating thoroughly without retaliating
- Communicating the outcome to the reporter

## Scope

This Code of Conduct applies to all project spaces — issues, pull requests, discussions, the official Discord (if/when it exists), commit messages, and any other communication channel maintained by the project — as well as in-person events (conferences, meetups) where someone is representing this project.

## Enforcement

Maintainers may take any action they deem appropriate, including warning, temporary or permanent removal from project spaces, and (where applicable) reporting to the relevant platform.

The full enforcement guidelines and rationale are described in the Contributor Covenant 2.1 linked above.

---

By participating in this project, you agree to abide by this Code of Conduct.
